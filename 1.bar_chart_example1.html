D3 Javascript Bar Chart Example #1

In this bar chart example, the Javascript code creates an HTML div element for each item in an array, and sets each div's width and the text inside the div to be equal to the array value. Unlike other examples, no SVG code is used here to keep it simple. Standard CSS is used to do the basic styling.

See Mike Bostock's D3 Bar Chart Tutorial for more info.
