D3 Javascript Bar Chart Example #0

In this bar chart example, the chart is created by manually writing SVG code for all bars. This example is meant as a reference for SVG syntax. In the subsequent examples, similar SVG code will be auto-generated by Javascript code, rather than being written manually.

See Mike Bostock's D3 Bar Chart Tutorial #2 for more info.
